## 🚀 How to Use the Website

### Option 1: Open Locally (Simplest)
1. **Double-click** on [`index.html`](index.html) to open it in your web browser
2. The website will work perfectly for viewing locally
3. You can bookmark it for easy access

### Option 2: Host Online (For Sharing)
To share the website with others or access it from anywhere:

#### Using GitHub Pages (Free & Easy)
1. Create a free GitHub account at [github.com](https://github.com)
2. Create a new repository (name it something like "birthday-valentines")
3. Upload all your files to the repository
4. Go to Settings → Pages
5. Select "Deploy from a branch" and choose "main"
6. Your website will be available at: `https://yourusername.github.io/birthday-valentines`

#### Using Netlify (Free & Simple)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your entire project folder onto their deploy area
3. Get an instant live website link to share

#### Using Vercel (Free & Fast)
1. Go to [vercel.com](https://vercel.com)
2. Sign up and connect your GitHub repository
3. Deploy with one click

## 📸 Adding Your Photos

1. **Prepare your photos**:

3. **Add them to the images folder**:

## 🔧 Troubleshooting

### Photos Not Showing?
- Check file names match exactly: `photo1.jpg`, `photo2.jpg`, etc.
- Ensure photos are in the [`images/`](images/) folder
- Try refreshing the page (Ctrl+F5 or Cmd+Shift+R)
- Check that image files aren't corrupted

### Website Not Loading?
- Make sure all files are in the same folder
- Try opening [`index.html`](index.html) directly in your browser
- Check that [`styles.css`](styles.css) and [`script.js`](script.js) are in the same directory

### Hosting Issues?
- Ensure all files are uploaded to your hosting service
- Check that file names match exactly (case-sensitive)
- Verify the main file is named [`index.html`](index.html)
