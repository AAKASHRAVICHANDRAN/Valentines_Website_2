# Contributing to Valentine's Day Interactive Website 💕

Thank you for your interest in contributing to this romantic interactive website project! We welcome contributions that make the experience even more magical.

## 🌟 Ways to Contribute

### 🐛 Bug Reports
- Report issues with animations, music playback, or mobile compatibility
- Include browser version, device type, and steps to reproduce
- Check existing issues before creating new ones

### 💡 Feature Suggestions
- New animation effects or transitions
- Additional interactive elements
- Mobile experience improvements
- Accessibility enhancements

### 🎨 Design Improvements
- CSS animations and effects
- Responsive design enhancements
- Color scheme variations
- Typography improvements

### 📱 Mobile Optimization
- Touch interaction improvements
- Performance optimizations
- Cross-browser compatibility fixes

## 🚀 Getting Started

1. **Fork** the repository
2. **Clone** your fork locally
3. **Add your photos/music** to test with real content
4. **Make your changes** in a feature branch
5. **Test thoroughly** on desktop and mobile
6. **Submit a pull request**

## 📋 Development Guidelines

### Code Style
- Use consistent indentation (2 spaces)
- Comment complex animations and interactions
- Keep CSS organized by page/component
- Use meaningful variable and function names

### Testing Checklist
- [ ] Test on Chrome, Firefox, Safari, Edge
- [ ] Test on mobile devices (iOS/Android)
- [ ] Verify music playback works
- [ ] Check all animations are smooth
- [ ] Ensure responsive design works
- [ ] Test with actual photos/music files

### Performance
- Optimize images before committing
- Keep audio files under 5MB
- Test loading times on slower connections
- Minimize CSS/JS when possible

## 🎵 Audio Guidelines

When contributing audio-related features:
- Support both WAV and MP3 formats
- Respect the 20-second auto-limit
- Test autoplay behavior across browsers
- Consider mobile autoplay restrictions

## 🖼️ Image Guidelines

For photo-related contributions:
- Support common formats (JPG, PNG, WebP)
- Provide fallbacks for missing images
- Optimize for various screen sizes
- Test with different aspect ratios

## 📝 Documentation

- Update README.md for new features
- Add comments for complex code
- Include setup instructions for new dependencies
- Document any breaking changes

## 🔄 Pull Request Process

1. **Create a descriptive title**
2. **Explain the changes** and why they're needed
3. **Include screenshots/videos** for visual changes
4. **Test on multiple devices** and browsers
5. **Update documentation** if needed

## 💕 Code of Conduct

This is a project about love and celebration! Please:
- Be respectful and kind in all interactions
- Focus on constructive feedback
- Help create a welcoming environment
- Remember this is meant to spread joy

## 🎯 Priority Areas

We're especially interested in contributions for:
- **Mobile experience improvements**
- **Accessibility features**
- **Performance optimizations**
- **Cross-browser compatibility**
- **New romantic animations/effects**

## 📞 Questions?

Feel free to open an issue for:
- Questions about the codebase
- Ideas for new features
- Help with setup or development
- General feedback

## 🏆 Recognition

Contributors will be acknowledged in the README and project documentation. Thank you for helping make love stories more interactive and magical! 💕
